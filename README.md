# Web帳票 自動ダウンロード LP

CoreWorks のアプリ「**Web帳票 自動ダウンロード**」のランディングページ。

取引先ごとのサイトに毎日ログインして請求書・納品書を集める作業を、パソコンが自動で代行。
30以上のサイトを自動巡回し、新規分だけをダウンロード・整理して指定フォルダに保存する Windows 常駐ツールを紹介する。

## 公開URL
https://nsmtysak.github.io/webchohyo-lp/

## 構成
- `index.html` — 単一ファイルのLP（HTML/CSS/JSすべて内包）。画像は使わず、CSSで作り込んだモックアップ（自動巡回ウィンドウ・フォルダ・通知）で構成。
- CoreWorks ブランド配色（濃紺 `#1B3A5B` / オレンジ `#E2641B`）でカタログサイトと世界観を統一。

## 訴求トーン
Before（毎朝の「ログイン地獄」＝生々しい現場の痛み）→ After（操作0分・フォルダに揃っている）→ 理念（本質的な仕事に集中）に着地。
